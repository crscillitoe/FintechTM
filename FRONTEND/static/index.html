Installing

Fortran.io is like PHP: FastCGI runs a script which outputs HTML strings

Using Ubuntu 14.04

sudo apt-get install nginx gfortran spawn-fcgi git libfcgi-dev sqlite

git clone https://github.com/mapmeld/fortran-machine.git

Use gfortran to compile flibs and other libraries, then:

spawn-fcgi -a 127.0.0.1 -p 9000 ./fortran_fcgi
